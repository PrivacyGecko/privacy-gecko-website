## Design System Integration

### Component Library
These OG images follow the Privacy Gecko Design System:

- **Color Palette**: gecko-green, dark-slate, pricko-gold, privacy-blue
- **Typography**: Inter font family (48-72px headings, 32-42px body)
- **Spacing**: 40-80px margins, consistent padding
- **Border Radius**: 8-25px for cards and badges
- **Icons**: Shield, lock, AI circuit, document, mobile, globe, eye
- **Shadows**: Subtle glows for emphasis (green/gold/blue)

### Accessibility Standards
- **WCAG 2.1 AA Compliance**: All text meets 4.5:1 contrast ratio minimum
- **Readable at Small Sizes**: Tested at 400px width (typical social thumbnail)
- **Color-Blind Friendly**: High contrast, doesn't rely solely on color for information
- **Clear Hierarchy**: Bold headings, distinct product names, clear status badges

## Developer Handoff

### Implementation Notes
1. All images use absolute paths: `/images/og-[product].png`
2. Next.js metadata API automatically serves these from `/public/images/`
3. No CDN configuration needed - images served from same domain
4. File names are lowercase with hyphens (SEO-friendly)
5. No special characters in filenames

### Code References
OG images are referenced in:
- `/src/app/products/[product]/page.tsx` - Dynamic product pages
