## Technical Assets

### Generation Scripts
Three Node.js scripts are available in `/scripts/`:

1. **generate-og-images.js** (Main script)
   - Generates all 10 images from scratch
   - Uses canvas library for programmatic image creation
   - Includes all brand colors and design system values

2. **recreate-large-images.js** (Optimization script)
   - Simplified versions of token and lock images
   - Reduces file size by using simpler gradients
   - Maintains visual quality while meeting <100KB requirement

3. **optimize-og-images.js** (Compression script)
   - Checks all image file sizes
   - Applies maximum PNG compression
   - Reports optimization results

### Regeneration Commands
```bash
# Install dependencies
npm install canvas

# Generate all images
node scripts/generate-og-images.js

# Optimize specific images
node scripts/recreate-large-images.js

# Check file sizes
node scripts/optimize-og-images.js
```
